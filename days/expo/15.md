## üìö Dictionary for Developers

*   **Review of Previous Terms:**
    *   **Interface (TypeScript):** Defines the shape of an object.
    *   **Type Alias:** A new name for any type.
    *   **Union Type:** Allows a variable to be one of several types.
    *   **Intersection Type:** Combines multiple types into one.
    *   **`map()`:** Transforms each element in an array.
    *   **`filter()`:** Creates a new array with elements that pass a test.
    *   **`reduce()`:** Reduces an array to a single value.
    *   **`find()`:** Returns the first element that satisfies a test.
    *   **`some()`:** Checks if at least one element satisfies a test.
    *   **`every()`:** Checks if all elements satisfy a test.
