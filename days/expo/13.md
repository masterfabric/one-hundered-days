## üìñ Task Explanations

Today, you'll unlock the power of functional programming in JavaScript by mastering three essential array methods: `map`, `filter`, and `reduce`. These methods allow you to transform and manipulate arrays in a declarative and highly efficient way, often replacing traditional `for` loops with more concise and readable code.

`map` is perfect for transforming each item in an array into something new. `filter` is your go-to for selecting a subset of items based on a condition. `reduce` is the most versatile, allowing you to boil down an entire array into a single value, whether it's a sum, an object, or anything else. Learning to chain these methods together will enable you to perform complex data operations with remarkable elegance.

## üìö Dictionary for Developers

*   **`map()`:** An array method that creates a new array populated with the results of calling a provided function on every element in the calling array.
*   **`filter()`:** An array method that creates a new array with all elements that pass the test implemented by the provided function.
*   **`reduce()`:** An array method that executes a reducer function on each element of the array, resulting in a single output value.
