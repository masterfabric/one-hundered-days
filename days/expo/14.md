## üìñ Task Explanations

Building on yesterday's array methods, today you'll expand your toolkit with `forEach`, `find`, `some`, and `every`. These methods provide powerful and declarative ways to interact with arrays, making your code cleaner and more expressive than traditional `for` loops.

`forEach` is ideal when you simply need to perform an action for each element, like logging them to the console. `find` is perfect for quickly locating a specific element based on a condition. `some` and `every` are excellent for checking conditions across an entire array, determining if at least one element or all elements, respectively, meet certain criteria. These methods are frequently used in React Native for data processing and UI rendering logic.

## üìö Dictionary for Developers

*   **`forEach()`:** An array method that executes a provided function once for each array element.
*   **`find()`:** An array method that returns the value of the first element in the array that satisfies the provided testing function. Otherwise, `undefined` is returned.
*   **`some()`:** An array method that tests whether at least one element in the array passes the test implemented by the provided function. It returns a Boolean value.
*   **`every()`:** An array method that tests whether all elements in the array pass the test implemented by the provided function. It returns a Boolean value.
