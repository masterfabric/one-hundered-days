## üìö Dictionary for Developers

*   **Review of Previous Terms:**
    *   **Functional Component:** A JavaScript function that returns JSX.
    *   **Props:** Data passed from parent to child components.
    *   **`useState` Hook:** Manages component-specific state.
    *   **`useEffect` Hook:** Manages side effects and lifecycle events.
    *   **TypeScript Interface:** Defines the shape of objects and props for type-safety.
