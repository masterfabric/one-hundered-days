## üìñ Task Explanations

Flexbox is the cornerstone of layout in React Native. It provides a powerful and efficient way to arrange, align, and distribute space among items within a container, even when their size is unknown or dynamic. Unlike traditional CSS Flexbox, React Native's Flexbox is always enabled by default for `View` components, making it the standard for building layouts.

Today, you'll grasp the fundamental concepts: `flexDirection` determines if your items flow horizontally (`row`) or vertically (`column`). `justifyContent` handles how items are spaced out along that main axis, while `alignItems` controls their alignment along the perpendicular cross axis. Mastering these properties is crucial for building any responsive UI in React Native.

## üìö Dictionary for Developers

*   **Flexbox:** A layout model that allows items within a container to be arranged and distributed space efficiently, even when their size is unknown or dynamic.
*   **`flexDirection`:** A Flexbox property that defines the main axis of the flex container, establishing the direction in which flex items are placed within the flex container.
*   **`justifyContent`:** A Flexbox property that defines the alignment of items along the main axis of the flex container.
*   **`alignItems`:** A Flexbox property that defines the alignment of items along the cross axis of the flex container.
