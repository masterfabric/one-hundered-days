# Day 2: Understanding the Agentic Mindset: LLMs, Tools, and Memory

## 📝 Today's Agenda

*   **The Role of Large Language Models (LLMs):** The "brain" of the agent.
*   **Tool Use:** How agents can interact with the outside world through APIs and other tools.
*   **Memory:** The importance of short-term and long-term memory for agents.
*   **The Agent Loop:** The basic cycle of an agent: observe, think, act.

## 🚀 Today's Goal

By the end of today, you will understand how LLMs, tools, and memory work together in an agentic system, and how the observe–think–act loop drives agent behavior.

## 🤖 Prompt Examples

*   **Explain it to me like I'm 5:** "How do LLMs act as the 'brain' of an AI agent?"
*   **Scenario-based:** "Explain the concept of 'tool use' for AI agents with an example."
*   **Deep Dive:** "Why is memory important for an agent, and what are the different types of memory?"

## 💡 Today's Dictionary

*   **LLM (Large Language Model):** The reasoning core of an agent; it interprets context, plans actions, and generates responses.
*   **Tool Use:** The ability of an agent to call external APIs, functions, or services to act beyond plain text generation.
*   **Memory:** Stored context that lets an agent retain information across steps—short-term (session) and long-term (persistent).
*   **Agent Loop:** The recurring cycle of observing input, reasoning about the next step, and taking an action until the task is done.
