## ğŸ“– Task Explanations

Sometimes, you don't want to merge an entire branch into another, but you do want to pick out a single commit from that branch and apply it to your current branch. This is what `git cherry-pick` is for. It allows you to take any commit from anywhere in your repository and apply it to your current location. This can be useful for backporting a bug fix to a release branch, for example.
