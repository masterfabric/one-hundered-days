## ğŸ“– Task Explanations

Everyone makes mistakes. Knowing how to undo them is an important Git skill. There are two main ways to undo changes in Git: `git reset` and `git revert`. `git reset` is used to move the current branch to a different commit, which can be used to unstage files or to completely remove commits from the history. `git revert` creates a new commit that is the opposite of a previous commit. This is a safer way to undo changes that have already been shared with others.
