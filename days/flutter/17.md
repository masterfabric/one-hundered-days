## üìö Dictionary for Developers

*   **`main.dart`:** The entry point of a Flutter application. The `main()` function in this file is the first code that is executed when your app starts.
*   **`pubspec.yaml`:** The project's configuration file. It's used to manage dependencies (packages), fonts, images, and other assets.
*   **`MaterialApp`:** A widget that provides a number of features that are commonly required for Material Design applications, such as routing and theming.
*   **`Scaffold`:** A widget that provides a basic layout structure for a Material Design screen. It can have an app bar, a body, a floating action button, and more.
