## üìñ Task Explanations

Today is about making your programs "smart." Control flow statements allow your program to make decisions and repeat actions. `if/else` statements are the most basic form of decision-making. Loops (`for`, `while`) are fundamental for performing repetitive tasks, such as processing a list of items. `switch/case` statements are a clean way to handle multiple distinct cases for a variable's value, which can be more readable than a long chain of `if/else if` statements.

## üìö Dictionary for Developers

*   **Control Flow:** The order in which a program executes its statements. Control flow statements (like `if`, `for`, `while`) can alter this order.
*   **Conditional Statement:** A statement that performs different actions depending on whether a condition is true or false (e.g., `if/else`).
*   **Loop:** A control flow statement that allows a block of code to be executed repeatedly. `for` and `while` are common types of loops.
*   **Iteration:** A single execution of the code block within a loop.
*   **`switch` statement:** A control flow statement that compares the value of a variable to a list of cases and executes the code block associated with the matching case.
